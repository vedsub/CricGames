A collection of cricket-themed mini-games built with React + Firebase.

## Games

| Game | Description |
|------|-------------|
| **Box Cricket** | Multiplayer 3x3 grid game. Match players to categories. Real-time 1v1 with room codes. |
| **Tenaball** | Guess the Top 10 cricketers in a category. 3 lives, fuzzy name matching. |
| **Who Are You?** | Wordle-style guessing. Compare attributes like nationality, age, team, jersey number. |

## Tech Stack

- **Frontend:** React, Vite, Tailwind CSS
- **Backend:** Firebase Realtime Database
- **Data:** 586 IPL players from Cricsheet.org

## Quick Start

```bash
npm install
cp .env.example .env  # Add your Firebase keys
npm run dev
```

## Deploy

Works out-of-the-box with Vercel. Add Firebase env vars in dashboard.
